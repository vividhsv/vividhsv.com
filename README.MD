# vividhsv.com Personal Site Repo

### Prerequisite
 Make user you have Node, Gulp and Bower installed
 * [Nodejs](https://nodejs.org/en/)
 * [Gulpjs](http://gulpjs.com/)
 * [Bower](http://bower.io/)

### How to use
  * Install Prerequisite
  * Clone Repo
  * cd into Repo
  * ```npm install```
  * ```bower install```
  * ```gulp build```
